## Get Started
First, clone the repository by doing
``` bash
git clone https://github.com/FrancescoBrunelli/Hikin-Trento.git
```

Then install the dependencies for both server and client:
``` bash
# Inside Hikin-Trento/server/
npm install

# Inside Hikin-Trento/client/
npm install
```

Then start the web-app:
- Start the server by doing:
``` bash
# Inside Hikin-Trento/server/
npm run dev
```
- Start the front-end client by doing:
``` bash
# Inside Hikin-Trento/client/
npm run dev
```

After activating the client, connect to the provided localhost link (should be http://localhost:5173/)
